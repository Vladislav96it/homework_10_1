# HOMEWORK_10.1
## что тут:
- функцию, которая принимает на вход список словарей и значение для ключа 
state
 (опциональный параметр со значением по умолчанию 
EXECUTED
) и возвращает новый список, содержащий только те словари, у которых ключ 
state
 содержит переданное в функцию значение.
- функцию, которая принимает на вход список словарей и возвращает новый список, в котором исходные словари отсортированы по убыванию даты (ключ 
date
). Функция принимает два аргумента, второй необязательный задает порядок сортировки (убывание, возрастание).
